# 📮 연락처 폼 프로젝트

간단한 **연락처(Contact) 폼**를 HTML, CSS, JavaScript만 사용해 구현한 프로젝트입니다.
사용자는 이름, 이메일, 메시지를 입력할 수 있으며, **실시간 입력 유효성 검사**, **오류 메시지 표시**, **제출 성공 피드백** 기능을 제공합니다.

---

## 🎯 프로젝트 목표

- 시멘틱한 HTML 구조와 접근성을 고려한 폼 구현
- JavaScript를 활용한 입력값 유효성 검사 로직 구현
- 사용자 경험(UX)을 고려한 오류/성공 피드백 제공
- 외부 라이브러리 없이 순수 웹 기술로 기능 구현

---

## 🖥️ 데모 화면

> 이름, 이메일, 메시지를 입력하고 제출 버튼을 누르면 유효성 검사 결과에 따라 오류 메시지 또는 성공 메시지가 표시됩니다.

---

## ⚒️ 기술 스택

**HTML5**

- 시멘틱 태그 사용
- `label`, `required`, `aria-live` 등 접근성 고려

**CSS3**

- 반응형 레이아웃
- 호버 및 포커스 효과
- 오류 / 성공 상태 스타일링

**JavaScript (ES Module)**

- 입력 유효성 검사 로직 분리
- 모듈화된 구조로 유지보수성 향상

---

## 📋 주요 기능

✅ 이름 입력: 2자 이상 필수
✅ 이메일 입력: 올바른 이메일 형식 검사
✅ 메시지 입력: 10자 이상 필수
✅ 실시간 유효성 검사 및 오류 메시지 표시
✅ 모든 입력이 유효할 경우 제출 버튼 활성화
✅ 제출 시 성공 메시지 표시 후 폼 초기화

## 📁 프로젝트 구조

```
.

├── fonts
│   └── woff2              # Pretendard 폰트 파일
└── images
    └── contact-form.png   # 프로젝트 파비콘
```

## ▶️ 실행 방법

1. 프로젝트를 클론하고 클론한 프로젝트로 이동합니다.

```
git clone https://github.com/mimizae/vanilla-js-contact-form.git
cd vanilla-js-contact-form
```

2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 이름, 이메일, 메시지를 입력 후 **제출** 버튼을 클릭합니다.

> 별도의 서버나 빌드 과정 없이 바로 실행 가능합니다!

## ✨ 접근성 고려 사항

- 모든 입력 필드에 `label` 연결
- 오류 메시지에 `aria-live="polite"` 적용
- 키보드 접근 및 포커스 스타일 지원
